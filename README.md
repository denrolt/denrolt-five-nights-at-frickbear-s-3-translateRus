# 🎮 Руссификатор для *five nights at frickbear's 3*

Неофициальный руссификатор для игры *five nights at frickbear's 3*.  
выполнена частичная локилизация с использованием машинного перевода.

---

## 📥 Скачать

👉 **Последняя версия:**  
[Скачать руссификатор](https://github.com/denrolt/denrolt-five-nights-at-frickbear-s-3-translateRus/blob/main/Frickberars3_ModRus.zip)

---

## ⚙️ Установка

1. Сделайте резервную копию оригинальных файлов  
2. Скачайте архив  
3. Распакуйте в папку с игрой (с заменой файлов)  
4. Запустите игру  

---

## ℹ️ Особенности

- Тип перевода: машинный  
- Возможны ошибки и неточности  
- Подходит для понимания сюжета и интерфейса  

---

## ⚠️ Важно

- Это не официальный перевод для версии 1.0.5
- После обновлений игры может не работать  
- Используйте на свой страх и риск
